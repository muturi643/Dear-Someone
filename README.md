# Dear Someone 🖋️

**Bridge the gap from crushing to committing.**

Dear Someone is a human-centric AI tool designed to help you craft heartfelt, emotionally resonant letters. Whether you're checking in with a "Safety Net," making "The Move," or delivering a "Proposal," we help you say exactly what's in your soul.

![Dear Someone](https://images.unsplash.com/photo-1516414447565-b14be0adf13e?auto=format&fit=crop&q=80&w=1200&h=600)

## ✨ Features

- **Multi-Phase Discovery**: We share your core intent and that "Special Sauce"—the inside jokes and shared moments that make your bond unique.
- **AI-Powered Generation**: Using Gemini AI, we generate a "Compliment Shower" and a personalized letter draft.
- **Emotional Controls**: Fine-tune your letter with sliders for Vulnerability, Professionalism, and Nostalgia.
- **Emotional Intensity Gauge**: A visual indicator of your letter's emotional depth.
- **WhatsApp Optimized**: One-click sharing to WhatsApp, Twitter, and Facebook.
- **Digital Archive**: Save your best drafts to your personal letter collection.
- **Confetti Celebration**: Because telling someone how you feel is a win.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- A Google Gemini API Key

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables in a `.env` file:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS
- **Animations**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **AI**: Google Gemini API (@google/genai)
- **Effects**: Canvas Confetti
